# coper
Version:1.0.0

## 说明
1. 拷贝指定目录下既定格式指定的所有文件，并按其目录结构放于指定的新目录下；
2. 有‘#’的表示注释行（注意#必须放在开头）
3. 所有目录层级关系使用反斜线:\
4. 所有目录不经行嵌套；
5. 以下示例不包括单引号；

## 语法
1. 'root:'->资源根目录，注意以\结束，表示是个目录;
1. 'target:'->输出目录，注意以\结束，表示是个目录;
2. '|'->表示并列选择；
3. '~'->表示区间，区间判断目前不具备只能，就是简单的加法；
4. ':'->区间整体扩展名；

## 举例
1. 'A\B\'->A目录下的整个B目录中的文件，但不会遍历B里面的子目录，只是将B目录中的非目录文件执行拷贝；
2. 'A\b'->A目录下所有b.*文件
3. 'A\b.txt'->A目录下文件b.txt
4. 'A\{001~013}'->A目录下的文件001到013, 扩展名任意；
5. 'A\{001~013:doc}'->A目录下的文件001到013，扩展名必须为doc
6. 'A\{001|013|008|0113}'->A目录下对应的这些文件，扩展名任意；
7. 'A\{001~013:mp3|015.txt|008.png|0113}'->A目录下001到013的所有mp3文件，015.txt文件，008.png文件，0113.*
8. 'A\B\{:ani}'->A\B目录下所有扩展为ani的文件（仅限B目录，其子目录不会被遍历）
9. '{a.txt|A\|B\C\b.png}'->根目录（root目录）下的a.txt文件，A目录下的所有文件，B\C目录下的b.png文件；
